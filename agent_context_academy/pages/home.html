Карта обучения — по порядку

Темы расставлены в рекомендованной последовательности: каждая следующая опирается на предыдущие. Номер в углу = шаг.

🧠 Фаза 1 — Фундамент: как работает агент
1
✍️

Prompt Engineering

Zero-shot, Few-shot, CoT, Role Prompting, R-C-T-F-Q, Negative Constraints. Основа общения с LLM.

Зачем: получать нужный результат от LLM с первого раза, без лишних итераций.

A0фундамент 2
🧠

Context Engineering

Управление контекстным окном: signal vs noise, порядок слоёв, .cursorignore, Checkpoint, Caching. Уровень выше промптинга.

Зачем: управлять тем, что попадает в контекст, а не просто писать промпты.

A1архитектура 3
🧩

Контекст агента

Что такое context window, как агент видит мир, слои и приоритеты.

Зачем: понять, что реально видит модель и в каком порядке — без этого остальное не сложится.

A0фундаментобязательно 4
🔄

Agent Loop

Как агент работает изнутри: think → tool → observe. Turns, контекст, остановка.

Зачем: понять механику цикла — без этого непонятно почему агент иногда «зацикливается» или теряет инструкции.

A0механикафундамент 5
🔀

Explore → Plan → Code

Процесс работы с агентом после Agent Loop: контекст, план, код с проверкой. Параллель и fan-out — в связке с Subagents.

Зачем: применить цикл агента на практике — не прыгать сразу в правки без explore и approve плана.

A0процессс нуля 6
🎲

Почему модель ≠ модель

Temperature, sampling, harness, context quality. Claude в Cursor ≠ Claude в API.

Зачем: не удивляться, почему один и тот же промпт даёт разный результат.

Как проходить

1. Пойми слои

Начни с Prompt Engineering и Context Engineering, затем «Контекст агента» — слои и приоритеты. Дальше AGENTS/Rules.

2. Разбери приоритеты

Assessment проверяет не определения, а выбор приоритета при конфликте. Hooks > Rules > AGENTS > chat.

3. Закрепи кейсами

Пройди тренажёр, ответь на каверзные вопросы. Формула: источник → роль → ограничение → пример.
